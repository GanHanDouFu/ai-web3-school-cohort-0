# Learning Agent Prompt 记录

## 使用的工具

- **Claude Code** (Anthropic) - 主要学习 agent
- **模型**: Claude Opus 4.7
- **场景**: 课程资料整理、概念解释、笔记生成、代码生成

## Prompt 示例

### 1. 学习计划生成

```
根据 AI × Web3 School Week 1 学习大纲，帮我制定个人学习计划。
我的背景：AI 有一定基础（用过 ChatGPT/Claude），Web3 完全新手。
每日可用时间：2-3 小时。
请按优先级排列任务，先补 Web3 短板。
```

### 2. 概念解释

```
用简单的类比解释以下 Web3 概念：
- 钱包 vs 银行账户
- 签名 vs 登录确认
- Gas vs 手续费
- 智能合约 vs 后端 API
每个概念用一句话类比 + 一句话关键区别。
```

### 3. 合约代码生成

```
生成一个最简单的 Solidity 智能合约，功能：
- 存储一个 uint256 值
- 提供 set() 和 get() 函数
- 添加适当的安全检查
要求：代码可直接在 Remix IDE 编译部署。
```

### 4. 交互式学习产物

```
创建一个 Web3 基础概念测验网页，要求：
- 10 道选择题，涵盖钱包、签名、Gas、合约、测试网
- 每题有解释
- 显示最终得分
- 纯 HTML/CSS/JS，可直接浏览器打开
- 深色主题，现代 UI
```

## Agent 使用反思

### 做得好的
- 快速生成结构化笔记框架
- 概念解释清晰，类比恰当
- 代码生成质量可用，但需验证

### 需要注意的
- Web3 细节可能不是最新的（如 Gas 价格、测试网状态）
- 合约代码需要实际编译验证
- 不要盲目信任 AI 给的链接或数据
