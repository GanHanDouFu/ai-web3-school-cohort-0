概念解释
LLM 如何生成文本？

1. Token 化：输入文本被切分成 token（词、子词或字符）
2. 概率预测：模型根据上下文计算每个可能的下一个 token 的概率
3. 采样：根据 temperature 参数，从候选中选择一个 token
4. 拼接：选中的 token 加入上下文，重复步骤 2-3

Temperature：控制随机性。低 temperature（如 0.1）倾向选择最高概率 token，输出更确定；高 temperature（如 1.0）增加随机性，输出更多样。

注意：本页面使用模拟概率，不是真实模型输出。实际 LLM 的概率分布更复杂。
