# 个人学习计划

## 学员画像

- AI 新手 / Web3 新手 / 会基础脚本
- 目标方向：开发
- 每日投入：2-3 小时

## 学习路径

### 第一阶段：建立基础（第 1-2 周）

**AI 基础**
- [ ] LLM：大模型能做什么，不能替代什么
- [ ] Prompt：如何把任务目标、边界和输出格式写清楚
- [ ] Context：模型一次能看见什么，哪些信息可信
- [ ] RAG：如何把外部知识接入模型
- [ ] Agent：如何让模型进入工具调用和多步执行

**Web3 基础**
- [ ] Network：区块、共识、L2、RPC 和链上状态
- [ ] Cryptography：哈希、公私钥、签名
- [ ] Wallet：钱包是身份和签名入口
- [ ] Smart Contract：链上规则如何部署和调用
- [ ] Account Abstraction：Smart Account 如何表达 Agent 权限

### 第二阶段：交叉理解（第 3-4 周）

- [ ] Chain-aware Context：链上状态如何进入 Agent 上下文
- [ ] Web3 Tool Use：RPC、钱包、合约工具如何被 Agent 调用
- [ ] Agent Workflow：哪些步骤适合自动化
- [ ] Agent Wallet：Agent 能拿什么权限

### 第三阶段：项目实践（第 5 周起）

- [ ] 选择一个前沿探索方向做原型
- [ ] 参加 Hackathon 或提交项目

## 每日节奏

- 早上：查看今日任务，读 Handbook 章节
- 学习：2-3 小时
- 晚上：写 daily note，生成打卡草稿
